# Abstrak

Sistem e-commerce multiplatform untuk layanan pengiriman barang berbasis aplikasi telah berhasil dirancang dan dikembangkan pada penelitian ini. Sistem ini terdiri dari tiga platform utama yaitu Customer App dan Courier App yang dikembangkan menggunakan Flutter, serta Admin Panel yang dikembangkan menggunakan Svelte. Ketiga platform tersebut terintegrasi melalui backend berbasis Node.js dengan framework NestJS dan database PostgreSQL.

Penelitian ini mengintegrasikan berbagai API eksternal untuk mendukung fitur-fitur utama sistem. Payment gateway Midtrans diimplementasikan untuk memproses pembayaran non-tunai melalui berbagai metode seperti QRIS, Virtual Account, dan E-Wallet. Peta interaktif menggunakan OpenStreetMap dan library Leaflet ditampilkan pada Admin Panel dan Courier App. OpenRouteService digunakan untuk menghitung rute pengiriman, sementara WebSocket diimplementasikan untuk komunikasi real-time dalam update lokasi driver dan notifikasi pesanan.

Fitur auto-dispatch berhasil diimplementasikan dengan mengunakan formula Haversine untuk menghitung jarak antara driver dan lokasi penjemputan secara otomatis. Sistem secara otomatis menugaskan driver terdekat kepada pesanan baru berdasarkan jarak, sehingga mengoptimalkan waktu penjemputan dan pengiriman. Fitur pelacakan pesanan real-time memungkinkan customer memantau posisi driver di peta dan status pesanan mereka tanpa perlu melakukan refresh halaman. Selain itu, fitur manajemen stok dengan perhitungan Reorder Point dan fitur time slot booking berhasil memberikan kemudahan bagi admin dalam mengelola inventori dan memberikan fleksibilitas waktu pengiriman bagi customer.

Hasil pengujian menunjukkan bahwa seluruh fitur yang diimplementasikan berjalan sesuai dengan yang diharapkan. Hasil User Acceptance Test (UAT) terhadap 25 responden customer dan 3 responden courier menunjukkan nilai rata-rata yang baik, sehingga aplikasi dapat dinyatakan diterima oleh pengguna dan layak digunakan sebagai sistem e-commerce multiplatform untuk layanan pengiriman barang.

Kata kunci: E-commerce, Multiplatform, Flutter, Svelte, NestJS, Auto-dispatch, WebSocket
