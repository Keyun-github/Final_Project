## 5. Module vs Feature Definition (per user discussion)

**Module:** Used by 2+ applications (reusable)
**Feature:** Specific to 1 application

| Component | Category | Apps Used |
|-----------|----------|-----------|
| OpenStreetMap | Module | Admin + Courier |
| Location Tracking | Feature | Courier (send) → Admin (view) |
| Photo Storage | Feature | Courier (upload) → Admin (view) |
| Midtrans Payment | Feature | Customer App only |
| Auto-dispatch | Backend Logic | Internal |
| Authentication | Feature | Per-app login |
